NeeBo

❮ ❯

NeeBo is a Robot Car kit that uses an Arduino board to recieve an input and sends signals to the motors based on which signal came in.
A web server was created using Node.js and Express on a Raspberry Pi which uses a Serial Port node module to send data the Arduino board.
It has a simple web based UI with arrows and a live stream of Pi Camera using motion.
It also has the ability to be controlled using a Playstation Dualshock 4 controller with the dualshock-controller node module. The information from the dualshock controller is then sent over websockets.
NeeBo was built with HTML, CSS, JavaScript, jQuery, Node.Js, Express, Arduino IDE, WebSockets .

mySitter

A web application that allows parents to connect with potential babysitters within a certain radius. Utilizes swiping actions for matches or rejections of babysitters. After a match as been made, it will then make a list of matches for the parent to use and the parent can then contact the babysitter via messaging and begin negotiations.
mySitter was built with HTML, CSS, Bootstrap, JavaScript, jQuery, Node.Js, Express, Sequelize.

Mercedes Benz Event Parking

A web app that shows you events to Mercedes Benz Stadium in Atlanta, GA and the surrounding parking lots/costs on the same day.
Pulls data from Ticket Master API showing date, names and price ranges of event.
The date is then given to ParkWhiz's API, which then gives the coordinates to draw markers on Google Map's API to indicate parking prices and locations.
This web app was built with HTML, CSS, JavaScript, jQuery, and AJAX requests.
